# Shortcuts

## GNOME

`SUPER` Activities
`SUPER` + `L` Lock the screen
`ALT` + `F2` Run Command
`SUPER` + `D` Go to desktop and back

## Nautilus

`CTRL` + `1` List View
`CTRL` + `2` Icons View
`CTRL` + `L` Show directory path or type new location
`CTRL` + `H` Show hidden (toggle)
`CTRL` + `F` Toggle search view
`SHIFT` + `DEL` Delete permanently
