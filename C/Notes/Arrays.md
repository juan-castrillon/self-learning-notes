# Arrays in C

Collection of variables of the same data type

Declared with `[]` after the name
```c
int example[15];
int example2[] = {1,2,3,4,5};
```
To access a value
```
example[0]
```

> In C arrays are Zero numbered and have a fixed size
> Uninitialized arrays contain random values (garbage)
